Schedule Manager is a simple web-based application for managing an academic timetable.
The project allows users to add teachers, create lessons, and build a structured weekly schedule.
Main features:
Add and manage a list of teachers
Assign teachers to lessons
Create lessons with day, time, subject, and teacher
Display the full schedule in a table format
Support for all days of the week
Dynamic update of teacher selection
Simple and clear user interface

This project is designed as a basic educational timetable manager.
All data is handled on the client side unless extended with backend storage.
The interface focuses on simplicity and clarity, making it suitable for learning purposes or small personal projects.
